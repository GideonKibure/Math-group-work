QUESTION 01

1. Derive the following

a. Derivation of Compound Angle Formula

Definition: A compound angle is an angle formed by combining two simpler angles that are measured on different planes.

Consider the figure below:

Figure 1.1: Geometric construction for compound angle derivation

In the diagram, the triangles PQR, PRS, and PQT are all right-angled.

∠PRX = A, ∠XRQ = 90° - A, ∠XQR = A

Compound Angle for Sine

Consider,

$$\sin(A + B) = \frac{QT}{PQ}$$

But, QT = QX + TX

$$\sin(A + B) = \frac{TX + QX}{PQ}$$
$$\sin(A + B) = \frac{TX}{PQ} + \frac{QX}{PQ} = \frac{RS}{PQ} + \frac{QX}{PQ}$$

Since TX = RS

$$\sin(A + B) = \frac{RS}{PQ} \times \frac{PR}{PR} + \frac{QX}{PQ} \times \frac{QR}{QR}$$
$$\sin(A + B) = \frac{RS}{PR} \times \frac{PR}{PQ} + \frac{QX}{QR} \times \frac{QR}{PQ}$$

Using Trigonometry,

$$\frac{RS}{PR} = \sin A, \quad \frac{PR}{PQ} = \cos B, \quad \frac{QX}{QR} = \cos A, \quad \frac{QR}{PQ} = \sin B$$
$$\therefore \sin(A + B) = \sin A \cos B + \cos A \sin B$$
$$\therefore \sin(A - B) = \sin A \cos B - \cos A \sin B$$

Compound Angle for Cosine

Using the similar method as above,

$$\cos(A + B) = \frac{PT}{PQ}$$

But, PT = PS - ST

$$\cos(A + B) = \frac{PS - ST}{PQ} = \frac{PS}{PQ} - \frac{ST}{PQ} = \frac{PS}{PQ} - \frac{RX}{PQ}$$
$$\cos(A + B) = \frac{PS}{PQ} \times \frac{PR}{PR} - \frac{RX}{PQ} \times \frac{QR}{QR}$$
$$\cos(A + B) = \frac{PS}{PR} \times \frac{PR}{PQ} - \frac{RX}{QR} \times \frac{QR}{PQ}$$

Using trigonometry,

$$\frac{PS}{PR} = \cos A, \quad \frac{PR}{PQ} = \cos B, \quad \frac{RX}{QR} = \sin A, \quad \frac{QR}{PQ} = \sin B$$
$$\therefore \cos(A + B) = \cos A \cos B - \sin A \sin B$$
$$\therefore \cos(A - B) = \cos A \cos B + \sin A \sin B$$

Compound Angle for Tangent

Since,

$$\tan \theta = \frac{\sin \theta}{\cos \theta}$$
$$\tan(A + B) = \frac{\sin(A + B)}{\cos(A + B)} = \frac{\sin A \cos B + \cos A \sin B}{\cos A \cos B - \sin A \sin B}$$

Divide by \(\cos A \cos B\) in both numerator and denominator:

$$\tan(A + B) = \frac{\frac{\sin A \cos B}{\cos A \cos B} + \frac{\cos A \sin B}{\cos A \cos B}}{\frac{\cos A \cos B}{\cos A \cos B} - \frac{\sin A \sin B}{\cos A \cos B}}$$
$$\therefore \tan(A + B) = \frac{\tan A + \tan B}{1 - \tan A \tan B}$$
$$\therefore \tan(A - B) = \frac{\tan A - \tan B}{1 + \tan A \tan B}$$

b. Derivation of Half-Angle Formulas

Half-Angle Formula for Sine

Starting from the double-angle formula:

$$\cos 2\theta = 1 - 2\sin^2 \theta$$

Rearranging:

$$2\sin^2 \theta = 1 - \cos 2\theta$$
$$\sin^2 \theta = \frac{1 - \cos 2\theta}{2}$$

Let \(\theta = \frac{A}{2}\), then \(2\theta = A\):

$$\sin^2 \frac{A}{2} = \frac{1 - \cos A}{2}$$
$$\therefore \sin \frac{A}{2} = \pm \sqrt{\frac{1 - \cos A}{2}}$$

Half-Angle Formula for Cosine

From the double-angle formula:

$$\cos 2\theta = 2\cos^2 \theta - 1$$

Rearranging:

$$2\cos^2 \theta = 1 + \cos 2\theta$$
$$\cos^2 \theta = \frac{1 + \cos 2\theta}{2}$$

Let \(\theta = \frac{A}{2}\):

$$\cos^2 \frac{A}{2} = \frac{1 + \cos A}{2}$$
$$\therefore \cos \frac{A}{2} = \pm \sqrt{\frac{1 + \cos A}{2}}$$

Half-Angle Formula for Tangent

Using the quotient identity:

$$\tan \frac{A}{2} = \frac{\sin \frac{A}{2}}{\cos \frac{A}{2}}$$
$$\tan \frac{A}{2} = \frac{\pm \sqrt{\frac{1 - \cos A}{2}}}{\pm \sqrt{\frac{1 + \cos A}{2}}} = \pm \sqrt{\frac{1 - \cos A}{1 + \cos A}}$$

Alternative form:

$$\tan \frac{A}{2} = \frac{1 - \cos A}{\sin A} = \frac{\sin A}{1 + \cos A}$$

c. Derivation of Sine and Cosine Rules

Sine Rule:

In triangle ABC, drop altitude h from C to AB.

$$\sin A = \frac{h}{b}, \quad \sin B = \frac{h}{a}$$
$$\Rightarrow h = b \sin A = a \sin B$$
$$\therefore \frac{a}{\sin A} = \frac{b}{\sin B}$$

Similarly, by dropping altitude from B to AC:

$$\frac{a}{\sin A} = \frac{c}{\sin C}$$
$$\therefore \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$$

Cosine Rule:

Using the same triangle, by the Pythagorean theorem:

$$a^2 = (c - x)^2 + h^2$$

But \(h^2 = b^2 - x^2\) and \(x = b \cos A\):

$$a^2 = (c - b \cos A)^2 + (b^2 - b^2 \cos^2 A)$$
$$a^2 = c^2 - 2bc \cos A + b^2 \cos^2 A + b^2 - b^2 \cos^2 A$$
$$\therefore a^2 = b^2 + c^2 - 2bc \cos A$$

Similarly,

$$b^2 = a^2 + c^2 - 2ac \cos B$$
$$c^2 = a^2 + b^2 - 2ab \cos C$$

Summary of Results:

Formula Type Equation
Sine Rule \(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}\)
Cosine Rule \(a^2 = b^2 + c^2 - 2bc \cos A\)
Compound Angle (Sine) \(\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B\)
Compound Angle (Cosine) \(\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B\)
Compound Angle (Tangent) \(\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A \tan B}\)

QUESTION 03

3. Find the equation to the tangents and normal to the parabola \(y^2 = 16x\) at the points (16,16), (1, −4). The tangent intersect at the point T and the normal intersect at R. Prove that the line TR is parallel to the axis of the parabola.

Parabola: \(y^2 = 16x\)

Given: \(y^2 = 16x\)

Comparing with standard form \(y^2 = 4ax\):

$$4a = 16 \implies a = 4$$

Verification of points:

Point P(16,16): \(16^2 = 256\), \(16 \times 16 = 256\) ✓

Point Q(1,-4): \((-4)^2 = 16\), \(16 \times 1 = 16\) ✓

Equation of tangent at (x₁, y₁):

$$yy_1 = 2a(x + x_1)$$

Tangent at P(16,16):

$$16y = 8(x + 16)$$
$$16y = 8x + 128$$
$$y = \frac{x}{2} + 8 \quad \text{(Equation T₁)}$$

Slope m₁ = \(\frac{1}{2}\)

Tangent at Q(1,-4):

$$(-4)y = 8(x + 1)$$
$$-4y = 8x + 8$$
$$y = -2x - 2 \quad \text{(Equation T₂)}$$

Slope m₂ = \(-2\)

Intersection Point T of Tangents:

Equating T₁ and T₂:

$$\frac{x}{2} + 8 = -2x - 2$$
$$\frac{x}{2} + 2x = -10$$
$$\frac{5x}{2} = -10 \implies x = -4$$
$$y = -2(-4) - 2 = 6$$

Therefore, T = (-4, 6)

Equations of Normals:

For a parabola, slope of normal = \(-\frac{1}{\text{slope of tangent}}\)

Normal at P(16,16):

$$m_n = -\frac{1}{m_1} = -2$$
$$y - 16 = -2(x - 16)$$
$$y - 16 = -2x + 32$$
$$y = -2x + 48 \quad \text{(Equation N₁)}$$

Normal at Q(1,-4):

$$m_n = -\frac{1}{m_2} = \frac{1}{2}$$
$$y + 4 = \frac{1}{2}(x - 1)$$
$$y + 4 = \frac{x}{2} - \frac{1}{2}$$
$$y = \frac{x}{2} - \frac{9}{2} \quad \text{(Equation N₂)}$$

Intersection Point R of Normals:

Equating N₁ and N₂:

$$-2x + 48 = \frac{x}{2} - \frac{9}{2}$$

Multiply both sides by 2:

$$-4x + 96 = x - 9$$
$$-4x - x = -9 - 96$$
$$-5x = -105 \implies x = 21$$
$$y = -2(21) + 48 = -42 + 48 = 6$$

Therefore, R = (21, 6)

Line TR:

Points: T(-4, 6) and R(21, 6)

Since both points have the same y-coordinate, line TR is horizontal: \(y = 6\)

Axis of the parabola:

The parabola \(y^2 = 16x\) has its axis along the x-axis: \(y = 0\)

Since TR is horizontal and the axis is horizontal, TR is parallel to the axis of the parabola.

Proof Summary:

T(-4, 6) and R(21, 6) lie on line y = 6

Parabola axis: y = 0

Both are horizontal lines
∴ Parallel

QUESTION 04

4. Find the coordinates of the points in which the line \(x + 2y = c\) meets the parabola \(y^2 = 10x\). Find the value of \(c\) when this line is a tangent to the parabola.

Line and Parabola Intersection

Given: Line: \(x + 2y = c\) and Parabola: \(y^2 = 10x\)

Finding intersection points:

From the line equation: \(x = c - 2y\)

Substitute into parabola:

$$y^2 = 10(c - 2y)$$
$$y^2 = 10c - 20y$$
$$y^2 + 20y - 10c = 0$$

This is a quadratic in y. The intersection points exist when the discriminant ≥ 0.

For tangency (line touches parabola at one point):

Discriminant Δ = 0

$$\Delta = (20)^2 - 4(1)(-10c) = 400 + 40c = 0$$
$$40c = -400 \implies c = -10$$

When c = -10:

$$y^2 + 20y - 10(-10) = y^2 + 20y + 100 = 0$$
$$(y + 10)^2 = 0 \implies y = -10$$
$$x = c - 2y = -10 - 2(-10) = -10 + 20 = 10$$

Point of contact:

$$(10, -10)$$

Coordinates of intersection points (for general c):

Using quadratic formula:

$$y = \frac{-20 \pm \sqrt{400 + 40c}}{2} = -10 \pm \sqrt{100 + 10c}$$
$$x = c - 2y = c - 2(-10 \pm \sqrt{100 + 10c})$$
$$x = c + 20 \mp 2\sqrt{100 + 10c}$$

Therefore, intersection points are:

$$(c + 20 - 2\sqrt{100 + 10c}, -10 + \sqrt{100 + 10c})$$
$$(c + 20 + 2\sqrt{100 + 10c}, -10 - \sqrt{100 + 10c})$$

Ans:

When \(c = -10\), the line is tangent to the parabola at point \((10, -10)\)

QUESTION 05

5. Given the ellipse \(\frac{x^2}{25} + \frac{y^2}{16} = \frac{1}{2}\), find;

a. The eccentricity

b. Equation of directrix

c. The foci



Ellipse: \(\frac{x^2}{25} + \frac{y^2}{16} = \frac{1}{2}\)

Given equation:

$$\frac{x^2}{25} + \frac{y^2}{16} = \frac{1}{2}$$

Multiply both sides by 2:

$$\frac{x^2}{50} + \frac{y^2}{32} = 1$$

This is of the form \(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\) where:

$$a^2 = 50 \implies a = \sqrt{50} = 5\sqrt{2}$$
$$b^2 = 32 \implies b = \sqrt{32} = 4\sqrt{2}$$

Since \(a > b\), the major axis is along the x-axis.

a. Eccentricity

$$c^2 = a^2 - b^2 = 50 - 32 = 18 \implies c = \sqrt{18} = 3\sqrt{2}$$
$$e = \frac{c}{a} = \frac{3\sqrt{2}}{5\sqrt{2}} = \frac{3}{5}$$

b. Equation of Directrix

For an ellipse, the directrices are given by \(x = \pm \frac{a}{e}\):

$$x = \pm \frac{5\sqrt{2}}{\frac{3}{5}} = \pm \frac{5\sqrt{2} \times 5}{3} = \pm \frac{25\sqrt{2}}{3}$$

c. The Foci

The foci are at \((\pm c, 0)\):

$$(\pm 3\sqrt{2}, 0)$$

Summary Table:

Parameter Value
Semi-major axis (a) \(5\sqrt{2}\)
Semi-minor axis (b) \(4\sqrt{2}\)
Linear eccentricity (c) \(3\sqrt{2}\)
Eccentricity (e) \(\frac{3}{5}\)
Foci \((\pm 3\sqrt{2}, 0)\)
Directrices \(x = \pm \frac{25\sqrt{2}}{3}\)

QUESTION 06

6. A curve has equation \(9x^{2} + 25y^{2} = 225\). Show that, it is an ellipse and hence find;

a. The eccentricity

b. Coordinate of foci

c. The equation of directrix

d. The length of major axis and minor axis


Ellipse: \(9x^2 + 25y^2 = 225\)

Given: \(9x^2 + 25y^2 = 225\)

Divide both sides by 225:

$$\frac{9x^2}{225} + \frac{25y^2}{225} = \frac{225}{225}$$
$$\frac{x^2}{25} + \frac{y^2}{9} = 1$$

This is of the form \(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\), therefore it is an ellipse.

$$a^2 = 25 \implies a = 5, \quad b^2 = 9 \implies b = 3$$
$$c^2 = a^2 - b^2 = 25 - 9 = 16 \implies c = 4$$

a. The Eccentricity

$$e = \frac{c}{a} = \frac{4}{5}$$

b. Coordinates of Foci

$$(\pm c, 0) = (\pm 4, 0)$$

c. Equation of Directrix

$$x = \pm \frac{a}{e} = \pm \frac{5}{4/5} = \pm \frac{5 \times 5}{4} = \pm \frac{25}{4}$$

d. Length of Major and Minor Axes

$$\text{Major axis length} = 2a = 2 \times 5 = 10$$
$$\text{Minor axis length} = 2b = 2 \times 3 = 6$$

Summary Table:

Parameter Value
Semi-major axis (a) \(5\)
Semi-minor axis (b) \(3\)
Linear eccentricity (c) \(4\)
Eccentricity (e) \(\frac{4}{5}\)
Foci \((\pm 4, 0)\)
Directrices \(x = \pm \frac{25}{4}\)
Major axis length \(10\)
Minor axis length \(6\)

QUESTION 07

7. Find the foci and length of the ellipse given by the equation; \(\frac{(x - 1)^2}{9} + \frac{(y - 4)^2}{16} = 1\).

Ellipse: \(\frac{(x-1)^2}{9} + \frac{(y-4)^2}{16} = 1\)

This is an ellipse in standard form \(\frac{(x-h)^2}{b^2} + \frac{(y-k)^2}{a^2} = 1\) with vertical major axis since \(16 > 9\).

Centre: (h,k) = (1,4)

$$a^2 = 16 \implies a = 4 \quad \text{(major axis vertical)}$$
$$b^2 = 9 \implies b = 3 \quad \text{(minor axis horizontal)}$$
$$c^2 = a^2 - b^2 = 16 - 9 = 7 \implies c = \sqrt{7}$$

Foci:

For a vertical major axis, foci are at \((h, k \pm c)\):

$$(1, 4 \pm \sqrt{7})$$

Length of Axes:

$$\text{Major axis length} = 2a = 2 \times 4 = 8$$
$$\text{Minor axis length} = 2b = 2 \times 3 = 6$$

QUESTION 08

8. Find foci and directrix of the following; \(25x^{2} + 4y^{2} - 50x + 8y = 71\).

Ellipse: \(25x^2 + 4y^2 - 50x + 8y = 71\)

Group x and y terms:

$$25(x^2 - 2x) + 4(y^2 + 2y) = 71$$

Complete the squares:

$$25[(x-1)^2 - 1] + 4[(y+1)^2 - 1] = 71$$
$$25(x-1)^2 - 25 + 4(y+1)^2 - 4 = 71$$
$$25(x-1)^2 + 4(y+1)^2 = 71 + 25 + 4 = 100$$

Divide by 100:

$$\frac{(x-1)^2}{4} + \frac{(y+1)^2}{25} = 1$$

Centre: (1, -1)

Since \(25 > 4\), the major axis is vertical.

$$a^2 = 25 \implies a = 5, \quad b^2 = 4 \implies b = 2$$
$$c^2 = a^2 - b^2 = 25 - 4 = 21 \implies c = \sqrt{21}$$

Eccentricity:

$$e = \frac{c}{a} = \frac{\sqrt{21}}{5}$$

Foci:

For vertical major axis, foci are at \((h, k \pm c)\):

$$(1, -1 \pm \sqrt{21})$$

Directrices:

For vertical major axis, directrices are \(y = k \pm \frac{a}{e}\):

$$y = -1 \pm \frac{a}{e} = -1 \pm \frac{5}{\sqrt{21}/5} = -1 \pm \frac{25}{\sqrt{21}}$$
$$y = -1 \pm \frac{25}{\sqrt{21}}$$

Summary Table:

QUESTION 09

9. Find the distance between the foci, eccentricity and the length of latus-rectum of \(3x^{2} + 4y^{2} = 12\).

Ellipse: \(3x^2 + 4y^2 = 12\)

Divide by 12:

$$\frac{3x^2}{12} + \frac{4y^2}{12} = \frac{12}{12}$$
$$\frac{x^2}{4} + \frac{y^2}{3} = 1$$
$$a^2 = 4 \implies a = 2, \quad b^2 = 3 \implies b = \sqrt{3}$$
$$c^2 = a^2 - b^2 = 4 - 3 = 1 \implies c = 1$$

Distance between the foci:

$$\text{Distance} = 2c = 2 \times 1 = 2$$

Eccentricity:

$$e = \frac{c}{a} = \frac{1}{2}$$

Length of latus-rectum:

$$\text{Latus rectum} = \frac{2b^2}{a} = \frac{2 \times 3}{2} = 3$$

QUESTION 10

10. Find the equation of the tangent and normal to the ellipse \(5x^{2} + 3y^{2} = 137\) at the point in the first quadrant whose x-coordinate is 2.

Ellipse: \(5x^2 + 3y^2 = 137\)

Point in first quadrant with x-coordinate = 2

Substitute x = 2 into the ellipse equation:

$$5(2)^2 + 3y^2 = 137$$
$$5(4) + 3y^2 = 137 \implies 20 + 3y^2 = 137$$
$$3y^2 = 117 \implies y^2 = 39 \implies y = \sqrt{39} \quad (\text{positive in first quadrant})$$

Point P: \((2, \sqrt{39})\)

Equation of tangent at P:

For ellipse \(5x^2 + 3y^2 = 137\), the tangent at \((x_1, y_1)\) is:

$$5xx_1 + 3yy_1 = 137$$
$$5(2)x + 3(\sqrt{39})y = 137$$
$$10x + 3\sqrt{39}y = 137$$

Equation of normal at P:

Slope of tangent:

$$10x + 3\sqrt{39}y = 137 \implies 3\sqrt{39}y = 137 - 10x$$
$$y = \frac{137}{3\sqrt{39}} - \frac{10}{3\sqrt{39}}x$$
$$\text{Slope of tangent } m_t = -\frac{10}{3\sqrt{39}}$$

Slope of normal:

$$m_n = -\frac{1}{m_t} = -\frac{1}{-\frac{10}{3\sqrt{39}}} = \frac{3\sqrt{39}}{10}$$

Equation of normal using point-slope form:

$$y - y_1 = m_n(x - x_1)$$
$$y - \sqrt{39} = \frac{3\sqrt{39}}{10}(x - 2)$$

Summary:

Item Equation
Point of contact \((2, \sqrt{39})\)
Tangent \(10x + 3\sqrt{39}y = 137\)
Normal \(y - \sqrt{39} = \frac{3\sqrt{39}}{10}(x - 2)\)

QUESTION 12

12. Write down the equation to the tangents and the normal to the hyperbola \(9x^{2} - 4y^{2} = 36\) at the point \((4, 3\sqrt{3})\).

Hyperbola: \(9x^2 - 4y^2 = 36\)

Divide by 36:

$$\frac{9x^2}{36} - \frac{4y^2}{36} = \frac{36}{36}$$
$$\frac{x^2}{4} - \frac{y^2}{9} = 1$$

Check if point \((4, 3\sqrt{3})\) lies on the hyperbola:

$$9(4)^2 - 4(3\sqrt{3})^2 = 9(16) - 4(27) = 144 - 108 = 36 \quad \checkmark$$

Equation of tangent at \((x_1, y_1)\):

For hyperbola \(\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1\), the tangent at \((x_1, y_1)\) is:

$$\frac{xx_1}{a^2} - \frac{yy_1}{b^2} = 1$$

Here, \(a^2 = 4\), \(b^2 = 9\), and \((x_1, y_1) = (4, 3\sqrt{3})\):

$$\frac{4x}{4} - \frac{3\sqrt{3}y}{9} = 1$$
$$x - \frac{\sqrt{3}}{3}y = 1$$
$$3x - \sqrt{3}y = 3$$

Equation of normal at \((4, 3\sqrt{3})\):

Slope of tangent:

$$3x - \sqrt{3}y = 3 \implies \sqrt{3}y = 3x - 3$$
$$y = \frac{3}{\sqrt{3}}x - \frac{3}{\sqrt{3}} = \sqrt{3}x - \sqrt{3}$$
$$\text{Slope of tangent } m_t = \sqrt{3}$$

Slope of normal:

$$m_n = -\frac{1}{m_t} = -\frac{1}{\sqrt{3}} = -\frac{\sqrt{3}}{3}$$

Equation of normal using point-slope form:

$$y - y_1 = m_n(x - x_1)$$
$$y - 3\sqrt{3} = -\frac{\sqrt{3}}{3}(x - 4)$$

Summary:

Item Equation
Point of contact \((4, 3\sqrt{3})\)
Tangent \(3x - \sqrt{3}y = 3\)
Normal \(y - 3\sqrt{3} = -\frac{\sqrt{3}}{3}(x - 4)\)
